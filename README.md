# SiteStraeon

#### O QUE É O STRAEON

O Straeon é uma biblioteca online gratuita de fácil acesso para criadores independentes ou para pessoas que não possuem dinheiro para comprar livros antigos e caros. Com o Straeon, você pode ler seus livros favoritos no seu navegador, além da opção de você poder salvar o livro que quiser para ler depois.
